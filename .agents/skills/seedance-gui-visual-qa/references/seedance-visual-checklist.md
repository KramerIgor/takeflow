# Seedance Visual Checklist

- `Projects`, `Single Generation`, `History`, and `Queue` tab behavior remains coherent.
- Single Generation and Queue accept/display image, video and audio reference files as designed.
- Chips/cards show thumbnail or media badge, filename, and remove control.
- Long prompts and filenames do not break layout.
- `Name` is optional and visibly replaces Episode/Scene only in Single Generation scope.
- History shows single-generation runs only, not queue/batch/continuation tasks.
- `Edit prompt` does not start generation.
- `Regenerate` shows the paid warning and is not confirmed during setup-run.
- Mobile/narrow layout has no overlap or clipped critical controls.
